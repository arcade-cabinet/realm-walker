# Story Thread C: Raven Mysteries

**Version**: 1.0 **DRAFT**

## Overview

The Raven Mysteries thread (C Story) is the deepest and most enigmatic narrative layer. Ravens appear throughout the game as mysterious guides, observers, and harbingers of change. This thread explores the hidden truths of the world and serves as a meta-narrative commentary on the player's journey.

## Thread Structure

**Progress Range**: 0-N (Variable, discovery-based)
**Thread Type**: Non-linear mystery investigation
**Primary Theme**: Truth, observation, and hidden knowledge

## The Nature of Ravens

### What Ravens Represent
- **Observers**: Watch and record all events
- **Guides**: Appear when player needs direction
- **Harbingers**: Signal major changes
- **Truth-Keepers**: Know secrets of the world
- **Meta-Entities**: Aware of game structure

### Raven Behaviors
- Appear at key narrative moments
- Provide cryptic hints and warnings
- React to player choices across all threads
- Lead to hidden areas and secrets
- Bridge between story threads

## Discovery Structure

Unlike A and B threads, C thread is non-linear and discovery-based. Ravens appear based on:
- Player progress in A and B threads
- Specific flag combinations
- Hidden area exploration
- Puzzle solving
- Repeated visits to locations

## Raven Encounters

### Encounter Type 1: The First Raven (cStoryProgress: 0)
**Flag**: `c_raven_noticed`
**Trigger**: Player completes tutorial
**Location**: Any outdoor area
**Message**: "They are watching. Always watching."

**Behavior**:
- Raven perches nearby
- Makes eye contact with player
- Flies away when approached
- Leaves behind black feather

**Unlocks**:
- Ability to notice ravens
- "Raven Spotted" UI element
- Feather collection mechanic

### Encounter Type 2: The Speaking Raven (cStoryProgress: 1)
**Flag**: `c_raven_spoke`
**Trigger**: Collect 3 black feathers
**Location**: Village Square at night
**Message**: "Not all truths are meant to be seen."

**Behavior**:
- Raven speaks in riddles
- Asks philosophical questions
- Provides first major hint
- Vanishes mysteriously

**Unlocks**:
- Raven dialogue system
- Cryptic hint log
- Philosophical choice tracking

### Encounter Type 3: The Guardian Raven (cStoryProgress: 2)
**Flag**: `c_raven_guardian_revealed`
**Trigger**: Meet first guardian (A thread beat 1)
**Location**: Same location as guardian encounter
**Message**: "Power has a price. Will you pay it?"

**Behavior**:
- Observes guardian encounter
- Comments on player's worthiness
- Hints at consequences of power
- Leaves warning feather

**Unlocks**:
- Raven commentary on A thread
- Power consequence hints
- Hidden guardian lore

### Encounter Type 4: The Political Raven (cStoryProgress: 3)

```typescript
  // Check if raven should appear
  shouldSpawn(
    location: string,

    aProgress: number,
    bProgress: number
  ): boolean;
  
  // Get raven behavior for current context
  getBehavior(encounterType: string): RavenBehavior;
  
  // Track raven sightings
  recordSighting(location: string, timestamp: number): void;
}
```

## Discovery Mechanics

### Raven Tracking Journal
Players maintain journal tracking:
- Ravens spotted (with locations and times)
- Feathers collected (with messages)
- Riddles solved (with interpretations)
- Truths learned (with implications)

### Pattern Recognition
Game tracks player ability to:
- Notice ravens in background
- Connect raven appearances to events
- Interpret cryptic messages
- Solve complex puzzles

### Reward Structure
- Each discovery grants lore entry
- Sets of discoveries unlock encounters
- Complete understanding grants secret ending
- Meta-awareness changes dialogue options

## Emergent Narrative

### Player-Driven Discovery
- No forced C thread progression
- Players choose how deep to investigate
- Optional but rewarding
- Affects game understanding and ending

### Multiple Truth Layers
- Surface truth (what seems obvious)
- Hidden truth (what ravens reveal)
- Ultimate truth (meta-narrative layer)
- Player's truth (their interpretation)

### Replayability
- Different discoveries each playthrough
- New encounters based on different choices
- Hidden content rewards exploration
- Secrets remain even after completion
