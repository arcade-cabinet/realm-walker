# Quest Taxonomy - Three-Tier Structure

**Version**: 1.1  
**Status**: QUEST DESIGN FRAMEWORK  
**Created**: October 6, 2025  
**Updated**: January 8, 2025 - Added database categorization  
**Purpose**: Organize 200+ quests into meaningful tiers with persistent tracking

---

## The Three Tiers

### MAJOR QUESTS (Epic Chains)
**Purpose**: Define faction relationships, unlock leaders, shape world

**Characteristics**:
- 5-10 quests per chain
- 2-4 hours to complete
- Multiple cutscenes
- Permanent world changes
- Unlock faction leader for RAID
- Major reputation swings (+50 to +100)

**Count**: 13 major chains (one per faction)

**Examples**:
- "The 40-Day Siege" (Ironbound) - Thrain's adoption ceremony
- "The 48th Scar" (Verdant) - Prevent extinction with Thalindra
- "The Restoration Equations" (Radiant) - Prove Luminara's math
- "The Unchained" (Corsairs) - Earn Blackwave's respect through combat

---

### MESO QUESTS (Story Arcs)
**Purpose**: Recruit heroes, explore faction culture, personal stories

**Characteristics**:
- 3-5 quests per arc
- 30-60 minutes to complete
- Character development
- Unlock specific hero
- Moderate reputation (+25 to +50)

**Count**: 39 meso arcs (3 per faction for 3 heroes)

**Examples**:
- "The Warmth Cloak" (Veilwalkers) - Earn Umbra's trust, receive cloak
- "The Thrall's Question" (Crimson) - Meet happy thrall, question morality
- "The Perfect Frequency" (Ashen) - Learn death song from Mortis
- "The Flesh Engine" (Ossuary) - Witness biological furnace, horror + function

---

### MINOR QUESTS (World Flavor)
**Purpose**: Make world feel alive, provide variety, earn gold

**Characteristics**:
- 1-3 quests (simple)
- 5-15 minutes each
- Repeatable or one-time
- Small reputation (+5 to +15)
- Gold rewards
- World-building details

**Count**: 100-150 minor quests (scattered everywhere)

**Examples**:
- "The Refugee's Story" - Help individual NPC, hear their tale
- "The Deserter" - Meet faction defector, learn what they escaped
- "The Child's Drawing" - Kid draws sun they've never seen
- "Border Patrol" - Help guards, see faction tensions
- "The Cost of Magic" - Witness mage collapse from spell exhaustion

## Quest Flow Example: "The 48th Scar" (Major)

**Tier**: Major (Verdant Allied chain)  
**Duration**: 3-4 hours  
**Quests**: 7

**Quest 1: "The Scars"**
- Meet Thalindra in World Tree heart
- She shows you her bark-scars (47 extinctions)
- Each scar tells a story
- *"The Silverback Elk. The Moonflower Vine. The Crystal Beetles. 47 species. Gone forever."*
- Reward: Understanding, +10 rep

**Quest 2: "The Threatened"**
- Thalindra senses 48th extinction approaching
- A species of glowing moths (provide light in dark areas)
- Eclipsed Flame is burning their habitat
- *"I cannot bear a 48th scar. Please. Save them."*
- Reward: Quest chain unlocks, +15 rep

**Quest 3: "The Habitat"**
- Travel to moth habitat (border with Scorched Wastes)
- See the moths (beautiful, bioluminescent)
- See the fire spreading (Eclipsed expansion)
- Meet the moths' guardian (minor NPC)
- Reward: Location discovered, +10 rep

**Quest 4: "The Negotiation"**
- Try to negotiate with Eclipsed Flame
- Meet fire cultists, they refuse (fire purifies)
- Ignatius: *"All things must burn. It is the way."*
- Peaceful solution fails
- Reward: Understand Eclipsed philosophy, choice unlocks

**Quest 5: "The Defense"**
- Combat: Defend moth habitat from fire cultists
- 3 waves of Eclipsed attackers
- Moths help (provide light, distract enemies)
- Win: Moths saved. Lose: Habitat burns (can retry)
- Reward: Moths saved, +25 rep

**Quest 6: "The Relocation"**
- Moths can't stay (too close to Scorched Wastes)
- Help relocate to Verdant Sanctum
- Escort quest (protect moths during travel)
- Eclipsed ambush (they want revenge)
- Final battle to save the moths
- Reward: Moths safe in Verdant territory, +25 rep

**Quest 7: "The Gratitude"**
- Return to Thalindra
- Cutscene: She does NOT carve 48th scar
- *"You saved them. In 1,247 years, few have earned such gratitude. You are guardian of life now. I name you Lifewarden."*
- Moths now light parts of Verdant Sanctum (permanent world change)
- Reward: Thalindra available for RAID, +100 total rep, title "Lifewarden", moth companion

**Total**: 3-4 hours, deeply emotional, permanent world impact

## Quest Pacing

**Act 1 (Tutorial)**: 4 minor quests (2 hours)

**Act 2 (Exploration)**: 
- 2-4 major chains (8-16 hours)
- 6-12 meso arcs (6-12 hours)
- 20-40 minor quests (3-6 hours)
- **Total**: 17-34 hours

**Act 3 (Finale)**:
- Final major quests (2-3 hours)
- RAID events (2-3 hours)
- WAR event (2-3 hours)
- **Total**: 6-9 hours

**Grand Total**: 25-45 hours depending on playstyle

---

## Why This Structure Works

**Major quests**: Epic, memorable, define relationships  
**Meso quests**: Personal, character-focused, recruit heroes  
**Minor quests**: Flavor, variety, make world feel alive  

**Together**: Rich, varied, emotionally resonant experience

**Not**: Grinding, busywork, filler

**Every quest matters. Every quest tells a story.**

**That's the promise.**

## Database Implementation

### Quest Categories in Schema

```prisma
enum QuestCategory {
  TUTORIAL      // Riverwatch intro (4 quests)
  FACTION_INTRO // First meeting (13 quests)
  MAJOR         // Epic chains (13 quests)
  MESO          // Hero recruitment (39 quests)
  MINOR         // World flavor (100+ quests)
  SHARD         // Main story (12 quests)
  RAID          // Multi-faction (4-6 events)
  WAR           // Finale (1 event)
  NEUTRAL       // Always available (30-40 quests)
  RANDOM        // Dynamic encounters
}
```

### Quest Rewards Schema

```prisma
model QuestReward {
  id        Int    @id @default(autoincrement())
  questId   Int
  quest     Quest  @relation(fields: [questId], references: [id])
  
  // Reward Types
  reputation     Int?      // Amount to add
  factionId      Int?      // Which faction
  heroId         String?   // Hero to recruit
  itemId         String?   // Item to grant
  gold           Int?      // Currency
  
  // Special Rewards
  unlocksLeader  Boolean   @default(false) // For RAID
  worldChange    String?   // Permanent effect
  abilityId      String?   // New player ability
  titleId        String?   // "Lifewarden", etc
  
  @@index([questId])
}
```

### Dynamic Quest Generation

For random encounters and repeatable content:

```typescript
// Generate context-aware minor quests
async function generateMinorQuest(location: Territory, playerRep: FactionRep[]) {
  const questPool = await prisma.quest.findMany({
    where: {
      category: "MINOR",
      OR: [
        { factionId: location.controllingFactionId },
        { factionId: null }, // Neutral quests
        { 
          faction: {
            relationships: {
              some: {
                faction2Id: location.controllingFactionId,
                relationshipType: { in: ["allied", "neutral"] }
              }
            }
          }
        }
      ]
    }
  });
  
  // Weight by reputation and world state
  return selectWeightedQuest(questPool, playerRep, worldState);
}
```

### Quest State Machine

Track quest progression states:

```typescript
enum QuestStatus {
  LOCKED       // Prerequisites not met
  AVAILABLE    // Can be started
  ACTIVE       // In progress
  COMPLETED    // Successfully finished
  FAILED       // Failed (can retry)
  ABANDONED    // Player quit
  EXPIRED      // Time-limited, missed
}

// State transitions
const validTransitions = {
  LOCKED: ["AVAILABLE"],
  AVAILABLE: ["ACTIVE", "EXPIRED"],
  ACTIVE: ["COMPLETED", "FAILED", "ABANDONED"],
  FAILED: ["ACTIVE"], // Retry
  ABANDONED: ["ACTIVE"], // Resume
  COMPLETED: [], // Terminal state
  EXPIRED: [] // Terminal state
};
```

### Performance Optimization

Quest data is cached for runtime:

```typescript
// Load all quest definitions at startup
const questCache = new Map<string, Quest>();

// But progress is always fresh from DB
async function getQuestStatus(saveGameId: number, questId: string) {
  const progress = await prisma.questProgress.findUnique({
    where: { saveGameId_questId: { saveGameId, questId } }
  });
  
  return {
    definition: questCache.get(questId),
    progress: progress || { status: "LOCKED" }
  };
}
```
